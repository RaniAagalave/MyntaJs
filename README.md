Project Overview
The Myntra Clone is a web application that replicates the core features of the popular fashion e-commerce platform Myntra. Built using JavaScript and related web technologies, this project demonstrates a modern approach to building a responsive, interactive online shopping experience.
<br>
Technologies Used
HTML/CSS: For structuring and styling the application.
JavaScript: For implementing interactive features and handling logic
